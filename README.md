## Contributing

### Setting up a dev environment

Requirements:

- Visual Studio Code
- Docker Desktop
- WSL 2

Setup:
- Clone repo into WSL 2
- Starup project in VS Code
- Reopen in Dev Container
- Run `docker-compose up` to start services
